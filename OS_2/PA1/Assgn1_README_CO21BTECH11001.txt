Instructions to run code:

1. Use the flag -pthread while compiling
   i.e., g++ Assign1_Src_CO21BTECH11001.cpp -pthread
   
   Note: Ignore compilation warnings if any.
   
2. Make sure to given input text file as command line argument.
   i.e., ./a.out in.txt
   
3. After successful execution, a file named output.txt will be created in same directory as source file.
   This output file will contain the following fields:
   a. Time taken to execute in microseconds
   b. Value of pi, as estimated by the program
   c. Log, which contains the information about the points generated by different threads.
